http://sample.org:3000/main?id=HTML&page=12

http : 프로토콜(통신규칙), 사용자가 서버에 접속할 때 어떤 방식으로 통신할 것인가에 대한 부분
        Hyper Text Transper Protocol, 웹 브라우저와 웹 서버가 서로 데이터를 주고 받기 위해서 만든 통신 규칙
        File Transper Protocol, etc...

sample.org : host(domain name)
3000 : 포트 번호. 한 대의 컴퓨터 안에 여러대의 서버가 있을 수 있다. 클라이언트가 접속했을 때 어떤 서버로 접속할지.. (항구 번호)
        80번 포트 - 웹 서버는 80번 포트를 쓴다고 전 세계적으로 약속되어 있다. http를 통해서 접속했다면 웹서버에 접속했다는 말, 포트 번호 생략시 80으로 접속된다.(디폴트 값)

main : path. 서버 컴퓨터의 어떤 디렉터리의 어떤 파일인지.

id=HTML&page=12 : 쿼리스트링. 웹 서버에 데이터를 요청할 때 데이터를 담아 요청을 보낼 수 있음.
                    쿼리스트링의 시작은 '?'로, 값과 값은 '&'로 구분, 값의 이름과 값은 '='로 구분
